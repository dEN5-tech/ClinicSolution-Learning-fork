## [1.0.0] - 2023-10-27 (Поставьте текущую дату)

Первый стабильный релиз библиотеки.

### Добавлено (Added)

- Класс `Patient` с логикой расчета возраста.

- Метод `CalculateBMI` для расчета индекса массы тела.

- Unit-тесты на базе xUnit.

- Настроен CI/CD конвейер (GitHub Actions) для тестов и авто-деплоя документации.

- Автоматическая генерация API-справочника из XML-комментариев.

- Документация в формате MkDocs (включая ГОСТ-раздел).

### Исправлено (Fixed)

- Исправлена ошибка в расчете возраста для високосных лет.

Инструкция:

●Added: для новых функций.
●Changed: для изменений в существующих.
●Fixed: для исправленных багов.
●Removed: для удаленного.
