# API Reference: ClinicCore

## Класс: Patient

Представляет основные данные пациента и методы расчета медицинских показателей.

### Свойства

| Имя | Тип | Описание |

| :--- | :--- | :--- |

| `FullName` | `string` | Полное имя пациента. |

| `BirthDate` | `DateTime` | Дата рождения. |

| `Weight` | `double` | Вес в килограммах. |

| `Height` | `double` | Рост в метрах. |

### Методы

#### `CalculateAge()`

Вычисляет полный возраст пациента в годах.

* **Возвращает:** `int` — Количество полных лет.

---

#### `CalculateBMI()`

Вычисляет Индекс Массы Тела (ИМТ).

**Формула:** $$ I = \frac{m}{h^2} $$

* **Возвращает:** `double` — Значение ИМТ.

* **Исключения:**

* `DivideByZeroException` — возникает, если свойство `Height` <= 0.

!!! warning "Внимание"

    Перед вызовом метода убедитесь, что свойство `Height` заполнено корректно.
